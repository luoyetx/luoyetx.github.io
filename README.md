Blog
====

My blog source.

### Install

```
$ git clone --recursive git@github.com:luoyetx/luoyetx.github.io.git
$ cd luoyetx.github.io
$ pip install -r requirements.txt
```

### Build

```
$ make html
$ make serve
```

### Deploy

deploy and publish via [Travis CI](https://travis-ci.org/luoyetx/luoyetx.github.io).
